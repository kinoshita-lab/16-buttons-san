# 16-buttons-san

## PCB with 16 buttons and shift registers

"16-buttons-san" is a handy PCB with 16 buttons, and shift resistors and can also cascade next same PCB.

## Kit contents

- Pre-assembled circuit board
- 1x5 pin socket
