# 16ボタンさん

## シフトレジスタ付きの16個のボタンの基板

16ボタンさんはシフトレジスタで構成された便利な基板です。別の基板にカスケード接続することもできます。
ピン数が少ないマイコンボードでも、ピン3本でスイッチを(処理できる範囲で)いくらでも増やすことができます。

## キットに含まれているもの

- 組みたて済みの基板
- 1x5 ピンソケット
